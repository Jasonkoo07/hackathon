# How to Run (For Team Members)
1. Press the green **[<> Code]** button.
2. Select **Download ZIP**.
3. Unzip the downloaded folder.
4. Run **main.py** in the folder (not **index.html**).
5. Go to the link (e.g., http://127.0.0.1:5000) that appears in the execution window. If you run it with IDLE and it doesn't connect to the link, try running it on the terminal.
